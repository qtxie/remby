# Plan: Context-Aware Search

## Goal
When searching from ContinueWatching/LatestItems, do local filtering. When searching from LibraryBrowser, use Emby API with ParentId.

## Current State
- `/` key only works in the default `_ =>` branch (Home, Episodes, SeriesInfo, Favorites)
- `search()` method always searches server-wide
- ContinueWatching/LatestItems/LibraryBrowser views have no search support

## Changes

### 1. `src/app.rs` — Add search context tracking

Add enum and field:
```rust
pub enum SearchContext {
    ServerWide,
    LocalHome,      // filter home_items locally
    Library(String), // search via Emby API with ParentId
}
```
Add `search_context: SearchContext` field to `AppState`.

Update `start_search()` to accept context:
```rust
pub fn start_search(&mut self, context: SearchContext) {
    self.searching = true;
    self.search_query.clear();
    self.search_results.clear();
    self.search_context = context;
}
```

### 2. `src/emby.rs` — Add search_with_parent method

```rust
pub async fn search_in_library(&self, query: &str, parent_id: &str) -> Result<Vec<MediaItem>>
```
Same as `search()` but adds `ParentId` query param.

### 3. `src/main.rs` — Wire up search in new views

- Add `/` key handler in LibraryBrowser branch: `state.start_search(SearchContext::Library(lib_id))`
- Add `/` key handler in ContinueWatching/LatestItems branch: `state.start_search(SearchContext::LocalHome)`
- Update existing `/` handler in `_ =>` branch: `state.start_search(SearchContext::ServerWide)`
- Update Enter search handler to dispatch based on `search_context`:
  - `ServerWide` → `client.search(&query)`
  - `LocalHome` → filter `home_items` by name containing query
  - `Library(id)` → `client.search_in_library(&query, &id)`

### 4. `src/ui.rs` — Update footer hints

Add `/` to footer for LibraryBrowser, ContinueWatching, LatestItems views.

## Files to modify
- `src/app.rs` — SearchContext enum, field, start_search signature
- `src/emby.rs` — search_in_library method
- `src/main.rs` — SearchLoaded handler + all `/` key handlers + Enter search dispatch
